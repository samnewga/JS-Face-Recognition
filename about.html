Program Information



Technologies Used:


  • JavaScript
  • TensorFlow.js
  • Face-Recognition.js
  • Barba.js


    • Breakdown:


    Tiny Face Detector is used to detect faces and then 68 landmark points are placed on the persons face. This 68 points are then processed through a face detection model that was trained using the widerface dataset.


    Test Results:


  • Mean Age Error = 4.54
  • Total Gender Accuracy: 95%


    • How to Run:


  • Enable webcam on browser.
  • Sit infront of webcam.
  • Double click the run button.
  • Get live results.
